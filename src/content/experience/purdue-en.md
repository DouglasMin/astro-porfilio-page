## Research Overview

Developed an innovative system that combines drone technology and AI to automatically detect and collect lost golf balls on golf courses.

## Core Technologies

- Real-time object detection using computer vision
- Golf ball detection using deep learning algorithms
- Autonomous UAV flight system implementation
- Real-time image processing and analysis pipeline development

## Research Achievements

- Developed high-precision object detection model
- Implemented autonomous flight path optimization algorithm
- Validated system in real golf course environments
