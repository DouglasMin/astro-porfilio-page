## 프로젝트 개요

퍼듀대학교(Purdue University)에서 진행한 연구 프로젝트로, 드론(UAV)과 인공지능 기술을 결합하여 골프장에서 분실된 골프공을 자동으로 탐지하고 수거하는 시스템을 개발했습니다.

## 핵심 기술

### 컴퓨터 비전 & 딥러닝
- YOLOv5 기반 실시간 객체 탐지 모델 개발
- 다양한 환경(잔디, 숲, 물가)에서의 골프공 인식률 95% 이상 달성
- 데이터 증강 기법을 활용한 모델 정확도 향상

### UAV 자율 비행
- 최적 비행 경로 알고리즘 설계 및 구현
- GPS 기반 자율 비행 시스템 통합
- 실시간 이미지 전송 및 처리 파이프라인 구축

### 시스템 통합
- 드론-AI 시스템 간 실시간 통신 구현
- 클라우드 기반 데이터 수집 및 분석
- 웹 대시보드를 통한 실시간 모니터링

## 성과

- 실제 골프장 환경에서 시스템 검증 완료
- 기존 수작업 대비 80% 시간 절감
- 연구 결과 학회 논문 제출

## 기술 스택

**AI/ML**: Python, TensorFlow, YOLOv5, OpenCV  
**Drone**: DJI SDK, MAVLink  
**Backend**: FastAPI, PostgreSQL  
**Cloud**: AWS S3, EC2
