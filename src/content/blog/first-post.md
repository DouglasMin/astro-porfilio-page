## Astro를 선택한 이유

포트폴리오 웹사이트를 만들면서 여러 프레임워크를 고려했지만, 최종적으로 **Astro**를 선택했습니다.

### 주요 장점

1. **빠른 성능**: Zero JavaScript by default
2. **View Transitions**: 매끄러운 페이지 전환
3. **Content Collections**: 타입 안전한 콘텐츠 관리
4. **Islands Architecture**: 필요한 곳에만 인터랙티비티 추가
